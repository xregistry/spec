# Samples

<!-- words: readme schemastore -->

This directory contains sample assets for the CloudEvents Registry model:

- [scenarios](scenarios): end-to-end sample registries that combine endpoints,
  messages, and schemas for representative domains and protocols.
- [schemas](schemas): schema-focused sample content, including a generated
  registry derived from the SchemaStore catalog.

If you are new to the CloudEvents composite model, start with
[scenarios/README.md](scenarios/README.md) and then inspect the corresponding
samples.
